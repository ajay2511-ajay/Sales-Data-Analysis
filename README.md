# Sales Data Analysis

This project analyzes sales data to identify trends, top-performing countries, and product categories.

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Key Insights
- Portugal is the top revenue-generating country
- Clothing is the highest-selling category
- Monthly sales trends were visualized

## Files
- Sales_Data_Analysis.ipynb: Jupyter notebook with full analysis
- Sales-Export_2019-2020.csv: Dataset used for analysis
